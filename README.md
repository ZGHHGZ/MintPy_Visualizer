# MintPy_Visualizer
实现MintPy所得时序形变结果的可视化展示。

使用本软件前，需先通过MintPy的save_qgis.py脚本生成shp数据。

本模块已嵌入ESIS中。（ESIS的处理过程中已自动生成shp文件，无需手动生成）    https://github.com/ZGHHGZ/ESIS
